Sistema de Gerenciamento de Tarefas

Um aplicativo web completo que permite aos usuários gerenciar suas tarefas diárias com recursos avançados de organização, lembretes e análise de produtividade.

Funcionalidades Principais

Este sistema permite que os usuários criem, organizem e acompanhem suas tarefas diárias. Os usuários podem categorizar tarefas, definir prazos, adicionar lembretes e marcar como concluídas. O dashboard principal exibe estatísticas de produtividade e gráficos de progresso.

O sistema inclui autenticação de usuários, sincronização em tempo real, notificações por e-mail para tarefas próximas do prazo e um design responsivo que funciona em qualquer dispositivo.

Tecnologias Utilizadas:

React Node.js MongoDB Express JWT Chart.js

Aplicativo de Clima

Um aplicativo web que fornece informações meteorológicas detalhadas e previsões do tempo para qualquer localidade do mundo.

Funcionalidades Principais

Este aplicativo detecta automaticamente a localização do usuário e exibe informações detalhadas sobre o clima atual, incluindo temperatura, umidade, velocidade do vento, pressão atmosférica e índice UV.

Os usuários podem pesquisar qualquer cidade do mundo, visualizar previsões para os próximos 7 dias, gráficos de variação de temperatura e receber alertas sobre condições climáticas extremas. O aplicativo também permite salvar locais favoritos.

Tecnologias Utilizadas:

JavaScript HTML5 CSS3 OpenWeather API Geolocation API Chart.js
